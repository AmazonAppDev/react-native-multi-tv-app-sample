# Product Overview

This is a React Native Multi-TV App Sample - a versatile TV application that demonstrates best practices for building cross-platform TV experiences.

## Key Features
- Multi-platform support: Android TV, Fire TV, tvOS, and Web
- Customizable left-side drawer navigation
- Grid layout for content selection with dynamic hero image header
- Detailed content screens with video player integration
- Efficient focus management optimized for TV remote controls

## Target Platforms
- **Android TV / Fire TV**: Primary TV platforms with remote control navigation
- **tvOS**: Apple TV platform with Siri Remote support
- **Web**: Browser-based version for testing and development

## Core User Experience
The app follows TV-first design principles with:
- Remote control navigation using directional pad (D-pad)
- Focus management for seamless navigation between UI elements
- Large, touch-friendly interface elements optimized for 10-foot viewing
- Video playback with custom controls designed for TV remotes
